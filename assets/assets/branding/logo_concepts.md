# TGTC Logo Concepts

These are proposal assets only. The live site is still using the current mark until one direction is chosen.

## 1. Signal Graph
- File: `assets/branding/tgtc_logo_concept_signal.svg`
- Positioning: closest to the current identity, but cleaner and more deliberate.
- Feel: premium tech, NetSuite extension, integration graph, automation.
- Best if you want continuity with the current site.

## 2. Platform Frame
- File: `assets/branding/tgtc_logo_concept_frame.svg`
- Positioning: more enterprise and architecture-focused.
- Feel: systems, modules, platform building, implementation rigor.
- Best if you want the brand to look more consulting-led than startup-led.

## 3. Orbit Core
- File: `assets/branding/tgtc_logo_concept_orbit.svg`
- Positioning: more AI-forward and modern.
- Feel: intelligence layer, orchestration, connected services.
- Best if you want the AI angle to read immediately.

## 4. Monoline Grid
- File: `assets/branding/tgtc_logo_concept_monoline.svg`
- Positioning: the most minimal and editorial option.
- Feel: precise, modern, understated, technical.
- Best if you want a more restrained high-end identity.

## Recommendation
- Best balance for TG Tech Consulting: `Signal Graph`
- Most premium enterprise look: `Platform Frame`
- Most differentiated AI direction: `Orbit Core`
